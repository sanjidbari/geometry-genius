What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model in web development. It is a programming interface for web pages that enables the dynamic manipulation and updating of a web page's content and structure.The DOM, to put it simply, is a structure that resembles a tree and represents the elements and content of a web page. The DOM tree's nodes represent each element on the page, including headings, paragraphs, images, and buttons.The DOM's goal is to make it possible for programmers to make dynamic, interactive web pages. Developers may alter the look, behavior, and content of a web page in response to user activities by gaining access to and modifying the components and content of the page through the DOM.

How will you select HTML elements using DOM? Name the DOM methods.

To select HTML elements using DOM, there are several methods available that allow you to access and manipulate elements in different ways. Here are some of the most commonly used DOM methods for selecting HTML elements:
getElementById(): This method allows you to select an element with a specific id attribute. Example: var element = document.getElementById("myElement");
getElementsByClassName(): This method allows you to select elements with a specific class name. Example: var elements = document.getElementsByClassName("myClass");
getElementsByTagName(): This method allows you to select elements by their tag name. Example: var elements = document.getElementsByTagName("p");
querySelector(): This method allows you to select the first matching element using a CSS selector. Example: var element = document.querySelector("#myElement .myClass");
querySelectorAll(): This method allows you to select all matching elements using a CSS selector. Example: var elements = document.querySelectorAll("p.myClass");

What is event bubble?

The Document Object Model (DOM) uses a concept called event bubbling to describe how events spread throughout the DOM tree hierarchy. An event, such as a mouse click or keypress, first triggers on an element before "bubble-up" across its parent elements until it reaches the document's root. This means that if an event happens on an inner element, like as a button inside a div, it will first be triggered on the button before bubbling up through the div, the body, the document object, and eventually the window object.

What is a callback function and why will you use it?

A callback function in programming is a function that is performed inside another function after being supplied as an input to that function. Using a callback function serves the primary aim of allowing the calling code to continue running while the callback function completes a particular operation and then returns a result to the calling function. In asynchronous programming, when some processes, such data retrieval or API queries, may take longer to finish, callback functions are frequently utilized. A callback function is used to process the returned result rather than stopping programme execution while waiting for these activities to finish.
